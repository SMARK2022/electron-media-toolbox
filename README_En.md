## ğŸ“– Introduction

**SMARK Media Tools** is a local media management tool designed for photographers. It provides an efficient automated solution for organizing large batches of photos generated by continuous shooting and handling duplicate images:

1.  **Smart Grouping**: Automatically clusters burst sequences and similar scenes using HSV histogram similarity.
2.  **Aesthetic Scoring**: Integrates ZJU's **LAR-IQA** (No-Reference Image Quality Assessment) algorithm, accelerated by ONNX Runtime / DirectML, to automatically select the best photos in each group.
3.  **Quick Export**: Rapidly mark photos for retention or deletion, and export selected photos to your target folder in one click.

> **Performance**: ~3ms/frame for thumbnails, ~1s/frame for AI inference (GPU dependent).

## ğŸ“¥ Installation

### 1. For End Users (Recommended)

Download the latest installer from the [Releases](../../releases) page:

- Download **`Electron Media Toolbox.msi`**
- Simply double-click to install. No additional configuration is required.

### 2. For Developers (Source)

Requirements: Node.js and Python environments.

```bash
# 1. Clone the repository
git clone https://github.com/SMARK2022/electron-media-toolbox.git

# 2. Setup Backend (Conda/venv recommended)
cd python
pip install -r requirements.txt
python web_api.py  # Start the API server

# 3. Start Frontend (In a new terminal)
cd ..
npm install
npm run start
````

> **Note**: In dev mode, Electron connects directly to your local Python source code. When building the production version, the system compiles the backend to an EXE using Nuitka and packages it together.

## ğŸ“ Changelog

### v2.1.1 (2025-11-23)

  * **Inference Backend Migration**: Switched from `onnxruntime-gpu` to `onnxruntime-directml`, providing unified support for NVIDIA/AMD/CPU on Windows platform without requiring separate CUDA/cuDNN installation.
  * **Compatibility Improvements**: Enhanced Nuitka onefile exit process and signal handling to prevent orphaned temporary directories from incomplete cleanup during force termination.
